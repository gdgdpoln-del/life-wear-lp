# ライフウェア事業部 LP制作 仕様書

## 1. プロジェクト概要
*   **プロジェクト名**: ライフウェア事業部（Life Wear Division）ランディングページ制作
*   **目的**: 新規事業の立ち上げに伴い、事業内容（作業服、スーツ、イベント衣装、男性用着物）を魅力的に紹介し、お問い合わせを獲得すること。
*   **ターゲット**: 法人（制服・イベント）、個人（スーツ・着物）

## 2. デザインコンセプト
**「伝統を纏い、機能を活かす」**
老舗呉服店の伝統（和の心）と、現代のワークウェアに求められる機能性を融合させたデザイン。

*   **配色**:
    *   **勝色（Deep Navy）**: メインカラー。信頼、伝統、誠実さを表現。
    *   **黄金（Gold）**: アクセントカラー。高級感と和の華やかさを演出。
    *   **白練（Off-white）**: ベースカラー。清潔感と読みやすさを重視。
*   **フォント**:
    *   見出し: 明朝体（Noto Serif JP） - 品格、伝統
    *   本文: ゴシック体（Noto Sans JP） - 視認性、現代的機能美

## 4. 技術仕様
*   **言語・フレームワーク**:
    *   HTML5 / Tailwind CSS (CDN版)
    *   JavaScript (Vanilla JS): Intersection Observerによるフェードイン、スクロール連動、ハンバーガーメニュー制御。
*   **特記事項**:
    *   **グリッドレイアウト**: 事業内容カード内における、複数画像の複雑な組み合わせ（2分割や3分割）とレスポンシブ対応。
    *   **ロゴ制御**: スクロール位置に応じたロゴの反転・色相制御をJavaScriptで実装。
